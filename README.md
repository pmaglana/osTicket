<h1>Installing osTicket on a Windows 11 Virtual Machine</h1>
<b>What is osTicket</b>
</p>
osTicket is an open-source help desk and ticketing system that lets organizations manage customer support requests by converting emails, phone calls, or web form submissions into organized, trackable tickets.
</p>
<b>About this Project</b>
</p>
In this project we will install and configure osTicket on a virtual machine including it’s prerequisite files and dependencies.
</p>
<b>Environment & Technology Used</b>
</p>
- Microsoft Azure Virtual Machine</p>
- Microsoft Windows 11</p>
- Remote Desktop</p>
<b>Prerequisites</b>
</p>
•	osTicket-Installation-Files.zip</p>
•	PHPManagerForIIS_V1.5.0.msi</p>
•	rewrite_amd64_en-US.msi</p>
•	php-7.3.8-nts-Win32-VC15-x86.zip</p>
•	VC_redist.x86.exe</p>
•	mysql-5.5.62-win32.msi</p>
•	osTicket-v1.15.8.zip</p>
•	HeidiSQL</p>
